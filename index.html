PlayStation Access Controller
Profile Editor

This website lets you edit profiles (button mapping etc.) on a PlayStation Access controller without having to use a PS5. Ideally Sony would release an official tool for this, but they haven't so far.

You will notice that when you press the "profile" button on your Access controller while it's connected to a PC, the profile LED indicators don't work. But the button still cycles through the profiles, you just have to guess which profile is currently active.

If you edit the profiles on your Access controller using this website and then connect it to a PS5, the PS5 will overwrite the profiles with whatever profiles were configured on the console. If you want to use your Access controller with both a PS5 and a PC, edit the profiles on the PS5.

Depending on what game or software you use your controller with, the buttons might not show up as expected and might require remapping in game. Please note that the same thing happens when you configure the controller using a PS5. It's not the controller's or this website's fault, it's just sad reality.

Currently this website doesn't let you edit stick sensitivity and deadzone.

Currently this website only works with the controller connected over USB, not Bluetooth. But you can of course edit the profiles over USB and then play over Bluetooth.
